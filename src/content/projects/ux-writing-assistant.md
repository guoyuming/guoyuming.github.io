Figma plugin，将 Agent 的 UX writing 和 conversion copy writing skill 产品化：

- 人类 review copywriting 对着设计稿最有 context，效果好于对着文档 review
- 上次的 SKILL 要在 CLI 中重复复制文档、复制链接、输入命令、复制优化文案、粘贴回 Figma，这个工作最适合一键产品化
- Plugin 内置了转化为目标的 Conversion copywriting skill 和产品界面文案的 UX writing skill，接入自己 API key 即可

局限：这个 plugin 可以一键将文案提升到 75 分；至于更高水准，需要更多的 context 注入、用户画像理解、英语 native 语感、品牌/组织偏好和 writer 自我风格，当前 75 分对我亦是帮助

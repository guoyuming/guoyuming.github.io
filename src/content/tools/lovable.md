## What It Does

Lovable is an AI-powered app builder that generates full-stack web applications from natural language descriptions. It creates production-quality React applications with Supabase backend, authentication, and real-time features. You describe what you want, see a live preview, and iterate through conversation.

## Key Features

- **Natural Language to App**: Describe your app and get a working full-stack application
- **Live Preview**: See changes in real-time as you refine your prompts
- **Full-Stack Output**: React frontend + Supabase backend with auth and database
- **GitHub Sync**: Generated code is pushed to a repository you own
- **One-Click Deploy**: Publish to production with custom domains
- **Iterative Building**: Refine features, fix bugs, and add pages through chat

## Who It's For

Founders, product managers, and designers who want to rapidly prototype and ship functional web applications. Ideal for MVPs, internal tools, and testing product ideas before committing engineering resources. Also great for developers who want to skip boilerplate setup.
